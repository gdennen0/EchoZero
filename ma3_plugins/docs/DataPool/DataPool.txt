# DataPool Reference Documentation
# Source: grandMA3 Dump() command on DataPool()
# Date: [When this was captured]

## Overview

The DataPool is the root container for all data objects in grandMA3. It contains 15 child pools, including Timecodes, Sequences, Groups, Macros, and more.

## Object Information

**Name:** 1 'Default'
**Class:** Pool
**Path:** Root/ShowData/DataPools/1 'Default'

---

## Access Methods

### Basic Access

```lua
-- Access the default DataPool
local pool = DataPool()

-- Get the default pool by index
local pool = DataPool()[1]

-- Access via :Children() method
local pools = DataPool():Children()
local default_pool = pools[1]
```

### Accessing Child Pools

```lua
-- Access Timecodes pool
local timecodes = DataPool().Timecodes
-- or
local timecodes = DataPool()[1].Timecodes

-- Access Sequences pool
local sequences = DataPool().Sequences

-- Access Groups pool
local groups = DataPool().Groups

-- Get all child pools
local children = DataPool():Children()
for i = 1, #children do
    Printf("Pool " .. i .. ": " .. children[i].name)
end
```

### Iterating Child Pools

```lua
-- List all child pools
local pools = DataPool():Children()
for i = 1, #pools do
    local pool = pools[i]
    local name = pool.name or "Unknown"
    Printf("Pool " .. i .. ": " .. name)
end
```

```lua
local pool = DataPool()

-- Access via property name (direct)

local count = pool.count

-- Access via :Get() method

-- Access with role (returns string representation)
local name_str = pool:Get("NAME", Enums.Roles.Edit)
```

```lua

local children = DataPool():Children()

```

## Common Operations

### Checking Pool Existence

```lua
-- Check if DataPool exists
local pool = DataPool()
if pool then
    Printf("DataPool exists: " .. pool.name)
end

-- Check if Timecodes pool exists
local timecodes = DataPool().Timecodes
if timecodes then
    Printf("Timecodes pool exists")
end
```

### Dumping Pool Information

```lua
-- Dump entire DataPool
DataPool():Dump()

-- Dump specific child pool
DataPool().Timecodes:Dump()
```

### Listing All Pools

```lua
function ListAllPools()
    local pools = DataPool():Children()
    Printf("Total Pools: " .. #pools)
    Printf("")
    
    for i = 1, #pools do
        local pool = pools[i]
        local name = pool.name or "Unknown"
        local count = pool.count or 0
        Printf(string.format("[%02d] %s (Count: %d)", i, name, count))
    end
end
```
