Block Base Class (Block/block.py): An abstract class defining the structure and behavior of all blocks.
Specific Blocks (e.g., LoadAudioBlock): Subclasses that implement specific functionalities.
Controllers:
InputController: Manages input ports for a block.
OutputController: Manages output ports for a block.
CommandController: Handles commands for a block.
Project Class (Project/Project.py): Manages all blocks and their interconnections.
Utilities: Logging, prompting, etc.
